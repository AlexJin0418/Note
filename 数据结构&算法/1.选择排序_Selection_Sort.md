# 1.选择排序 Selection Sort

选择排序将从数组的第一个元素开始遍历

每一轮遍历向后寻找整个数组内最小的一个并与当前启示位置的元素进行交换达到排序的目的

选择排序时间复杂度为 `O(n^2)`

## C++ 代码实现



```C++
template<typename T>
void selectionSort(T arr[], int n) 
{
    for (int i = 0; i < n; i++)
    {
        int minIdx = i;
        for (int j = i + 1; j < n; j++)
        {
            if (arr[j] < arr[minIdx])
            {
                minIdx = j;
            }
        }
        swap(arr[i], arr[minIdx]);
    }
}
```
