<details>
  <summary>下列语句是否合法？ ival = v1 + v2; ; </summary>
  
  ``` 
  该语句是合法的。ival = v1 + v2; 表示一个赋值语句，而后面的分号 ; 表示一条空语句。
  ```
</details>

  <summary> 下列语句是否合法？ while (iter != svec.end()) ; ++iter; </summary>
  
  ``` 
  该语句不合法，while循环后额外的分号导致while循环空转，所以iter无法自增，则代码将无休止循环下去。
  ```
</details>
